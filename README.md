ebook-generator is a script that automatically scrapes and compiles stories from various websites.
It downloads all chapters, processes images, and saves them as HTML, TXT, or EPUB for offline reading.

Key Features:

- Scrapes all chapters from a single story URL
- Downloads and embeds images into content
- Custom scraping via CSS query selectors
- Automatically combines chapters into a complete file
- Supports multiple story sources (easily extensible)
- Exports to flexible formats (HTML, TXT, EPUB)
